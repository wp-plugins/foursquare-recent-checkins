A widget that show your 5 most recent checkins at FourSquare. And on a map too. It's marvelous!

== Description ==

Show your 5 most recent checkins at FourSquare with this widget, including name, your comment, 
and a map showing where the checkin was.

The map size can be customized: one size for the latest checkin. And a different size for the other four.

Customiziable through CSS. Make it look the way you want.
